## Project Overview

This repository is my complete Digital Literacy Portfolio submitted as part of
the CSE0001 course at VIT Bhopal University. As a Student Digital Ambassador,
I completed five tasks — one for each module of the course — covering digital
tools, professional online presence, coding platforms, email etiquette, and
cybercrime awareness.

All work is organised in a single GitHub repository as required by the
assignment brief. A full written project report is available in the
`report/` folder.

## Task 5 — Cybercrime Awareness Case Study & Prevention Guide
**Module 5 | 20 Marks**

**Cybercrime Type Chosen:** UPI / Online Payment Fraud

**Part A — Case Study (`casestudy.md`):**
A 250-word case study about a student named Priya who was scammed via a
UPI collect request while selling textbooks on OLX. Covers how the crime
happened step-by-step, who is targeted, and the consequences.

**Part B — Prevention Checklist (`prevention-checklist.md`):**
A 10-point Stay Safe Online checklist with 2 UPI-specific tips and
official reporting channels:

> 🌐 **Portal: cybercrime.gov.in**
